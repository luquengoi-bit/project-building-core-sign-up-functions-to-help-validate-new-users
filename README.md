# Validating user registrations (Python)

## Objective

Create Python functions to validate the information entered by users during registration.

## Analyses

- Checking the name format
- Checking the email format
- Validating user inputs

## Results

- Names must be strings of more than 2 characters
- Emails must contain ‘@’ and a valid domain
- Reduction in errors during registration
